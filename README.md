## 설치

각 환경에서 `VIM` 설치는 다음과 같다. (Windows는 무시한다.)

### CentOS

```bash
$ sudo yum install vim-enhanced
```

### Ubuntu

```bash
$ sudo apt-get install vim
```

### MacOS
